# ISA Prep Course Website

## Files

- `index.html` — course homepage
- `styles.css` — shared styling and sidebar navigation
- `tree-fertilization/index.html` — interactive fertilizer lesson
- `tree-fertilization/script.js` — answer checking for the fertilizer lesson
- `soil-analysis/index.html` — soil analysis lesson

## Upload to GitHub

Upload the complete folder structure to the root of the `ISA-Prep-Course` repository. Do not flatten the two lesson folders.

## Publish with GitHub Pages

1. Open the repository's **Settings**.
2. Select **Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and `/(root)` folder.
5. Save.

The site address will normally be:

`https://YOUR-USERNAME.github.io/ISA-Prep-Course/`
